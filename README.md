# RKD Common Maps

A simple Unreal Engine editor plugin that adds quick access to frequently used maps directly in the toolbar.

## Features

- 📌 Toolbar dropdown with categorized map list
- ➕ Add maps to the list via Content Browser context menu
- ➖ Remove maps from the list via Content Browser context menu
- 💾 Settings are saved automatically via Developer Settings

## Installation

1. Copy the `RKDCommonMaps` folder into your project's `Plugins/` directory
2. Restart Unreal Engine
3. Enable the plugin in **Edit → Plugins → RKD Common Maps**

## Usage

### Adding a map
1. Right-click any map asset in the Content Browser
2. Select **Common Maps → Add to Common Maps**
3. Choose a category (or create a new one in settings)

### Removing a map
1. Right-click the map asset in the Content Browser
2. Select **Common Maps → Remove from Common Maps**
3. Choose the category to remove it from

### Settings
You can manage categories and maps manually in:
**Edit → Editor Preferences  → RKD Common Maps**
